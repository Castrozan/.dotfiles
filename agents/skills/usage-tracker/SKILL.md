# Usage Tracker -- Claude Usage Dashboard

Displays Claude Code subscription usage stats including daily breakdown, weekly totals, and session counts.

## Usage

```bash
scripts/usage-tracker.sh
```

## Requires

- `ccost` CLI installed at `~/.local/bin/ccost`
- Claude Code stats cache at `~/.claude/stats-cache.json`
